Backgrounds o Fondos

Esta pagina contiene un archivo CSS que se llama backgrounds.css el cual puede ser consultado.

A continuacion algunas formas de poner background:

Background Color

para el ejemplo voy a crear un div, con unas dimensiones de alto 20% y de anho 70% donde lo pondre un color y texto dentro

Soy el div, se uso con la propiedad background-color

Backgorund Image

La propiedad background image, se esta usando como el fondo de esta pagina, se puso en el body. la propiedad es o se debe invocar bajo el nombre
background-image: url("nombredelaimagen.jpg")
esta se puso en el body, pero tambien la pudimos poner en un div, donde consideremos oportuno.
