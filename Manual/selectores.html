Selectores CSS

Los selectores hacen referencia a la forma como llamo un elemento para aplicarle un estilo, a continuacion muestro las maneras de hacerlo:

Por elementos:

Se puede seleccionar por el elemento que se esta usando, por ejemplo:
p{ text-align: center;
color: red;
}
Tambien es posible que se pueda llamar por varios elementos que se les quiera aplicar el mismo estilo, ej:
p, h1, h2{
color: red;
}

Por ID:

para seleccionar por id, primero se debe crear ese id en el html, de la siguiente manera:
<p> id="para1" </p> (en el html)
En el CSS se llamara de la siguiente manera:
#para1 {
text-align: center;
color: blue;
}

Por Clase:

Por clase, se debe crear en el html lo siguiente:
<p> class="para2" </p>
En el CSS se llamara de la siguiente manera:
.para2{
text-align: center;
color: yellow;
}
